# django-reading
読書メモアプリ

```
% python3 -V
Python 3.12.1
```
```
% git clone https://github.com/wanana12/django-reading.git
```
```
% cd django-reading
% python3 -m venv .venv
% source .venv/bin/activate
```
```
% pip install -r requirements.txt
% python -m django --version
5.0
```
```
% python manage.py migrate
% python manage.py runserver
```
